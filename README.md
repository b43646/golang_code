## gRPC ##
`calculator` 目录提供三种gRPC API类型示例,分别是:
- unary
- client streaming
- bi directional streaming

![Image of Yaktocat](https://github.com/b43646/golang_code/blob/master/api_type.png)

`helloworld` 目录提供了gRPC SSL配置示例。
